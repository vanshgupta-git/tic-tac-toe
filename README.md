## 📖 About The Project

This project is a fully functional implementation of the classic **Tic-Tac-Toe** game developed using **Vanilla JavaScript**. It features a modern light-themed user interface designed to provide a pleasant gaming experience while maintaining simplicity and responsiveness.

The game automatically detects:

- 🏆 Winners
- 🤝 Draws
- 🔄 Game Resets
- 🎮 New Matches

No page refreshes are required, ensuring a seamless and uninterrupted user experience.

## 🎮 How To Play

1. Player **O** starts the game.
2. Players take turns selecting an empty box.
3. The first player to align three marks horizontally, vertically, or diagonally wins.
4. If all boxes are filled and no winner is found, the match ends in a draw.
5. Click **New Game** or **Reset** to start a fresh match.
